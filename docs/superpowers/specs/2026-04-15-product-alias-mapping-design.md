# 产品别名映射整理工具设计

## 项目背景
当前已经合并生成了统一产品库 `product_unified.xlsx`，但由于不同客户发来的产品编码存在差异，同一个实际产品可能对应多个不同编码（别名）。需要整理出：
1. 纯净的标准产品库（每个产品一行）
2. 完整的别名→标准编码映射表
3. 方便查阅的标准产品带别名汇总表

## 设计目标
1. 按照指定规则自动从现有统一产品库中找出疑似别名对
2. 生成纯净的标准产品库（去除别名重复）
3. 生成规范的别名映射表
4. 生成带别名列表的汇总表方便查阅

## 别名识别规则

### 规则一：第五位1/2差异（规格相同）
- 判断条件：规格名称相同，去掉分隔符'-'后长度相同，其他位都相同，只有**第五位**一个是'1'一个是'2'
- 处理方式：第五位是**2**的作为别名，保留第五位是**1**作为标准编码

### 规则二：末尾T差异（规格相同）
- 判断条件：规格名称相同，一个编码末尾是'T'，去掉末尾'T'后与另一个编码完全相同
- 处理方式：末尾带'T'的作为别名，保留不带'T'作为标准编码

### 规则三：含*号尺寸标记（规格相同）
- 判断条件：编码中包含'*'号（表示尺寸，如 `*1.5`），去掉'*'及其后的尺寸部分后与另一个编码完全匹配
- 处理方式：带'*'号的作为别名，保留不带尺寸标记的作为标准编码

## 输出文件

| 输出文件 | 位置 | 格式说明 |
|---------|------|---------|
| `output/product_standard.xlsx` | output | 标准产品库，格式与原 `product_unified.xlsx` 相同，仅保留标准产品（每个产品一行），去掉别名行 |
| `data/alias_mapping.xlsx` | data | 别名映射表，每条别名一行 |
| `output/standard_with_aliases.xlsx` | output | 标准产品汇总，每个标准一行，带所有别名列表 |

### 输出字段说明

#### product_standard.xlsx
与原 `product_unified.xlsx` 格式完全一致：
| 列名 | 说明 |
|------|------|
| unified_code | 标准统一编码 |
| product_name | 产品名称 |
| spec | 规格 |
| unit | 单位 |
| cost_price | 成本价 |
| sale_price | 售价 |
| category | 分类 |
| source | 来源 |

#### alias_mapping.xlsx（保持原有表头）
| 列名 | 说明 |
|------|------|
| 别名 | 别名编码 |
| 统一编码 | 对应的标准统一编码 |
| 别名类型 | 规则一/规则二/规则三 |
| 来源 | 自动发现 |
| 创建时间 | 自动填入当前时间 |
| 备注 | 规格信息 |

#### standard_with_aliases.xlsx（新增汇总表）
| 列名 | 说明 |
|------|------|
| unified_code | 标准统一编码 |
| product_name | 产品名称 |
| spec | 规格 |
| sale_price | 售价 |
| 别名列表 | 所有别名，用逗号分隔（无别名留空） |
| 别名规则 | 对应别名的规则名称，逗号分隔 |

## 处理流程

```
┌────────────────────────────┐
│  读取当前 product_unified   │
└────────────┬───────────────┘
             │
             ▼
┌────────────────────────────┐
│  按规格分组                │
│  同规格内才可能是别名关系   │
└────────────┬───────────────┘
             │
             ▼
┌────────────────────────────┐
│  依次应用三条规则找别名    │
│  规则一 → 规则二 → 规则三  │
└────────────┬───────────────┘
             │
             ▼
┌────────────────────────────┐
│  确定标准编码和别名映射    │
└────────────┬───────────────┘
             │
             ▼
┌────────────────────────────┐
│  输出三个文件              │
│  1. product_standard.xlsx  │
│  2. 更新 alias_mapping.xlsx │
│  3. standard_with_aliases.xlsx │
└────────────────────────────┘
```
