# OpenClaw 自动报价技能配置与使用指南

您好！本指南将教您如何向您的 AI 助手（如 OpenClaw、Cursor 或本项目内置的 Agent）配置和使用刚刚创建的**“自动处理询价单”**技能。

## 1. 技能原理说明
在刚刚的操作中，我为您创建了两个核心文件：
1. **技能定义文件 (`.agents/workflows/auto-inquiry.md`)**: 这是给 AI 看的“说明书”。它告诉 AI：当收到您的询价文字时，如何去提取产品和数量，并将其转化为 JSON 格式。
2. **后台处理引擎 (`server/scripts/ai_auto_quote.js`)**: 这是给 AI 在系统底层执行的脚本。AI 将提取出来的 JSON 交给它，脚本会全自动穿透您的系统数据库（进行精准+别名匹配），并直接为您生成系统草稿报价单。

## 2. 如何配置 OpenClaw
在绝大多数现代 AI 开发助手（OpenClaw / Cursor 等）中，无需进行复杂的代码集成，您只需要确保 AI 能够**读取工作区特定文件夹即可**：

1. **确保存放路径正确**：
   技能文件已经保存在了您项目的 `.agents/workflows/auto-inquiry.md` 目录下。这是一个 AI 会默认扫描并识别为其自身技能工作流（Workflows）的通用文件夹结构。
   
2. **（可选）唤醒词配置**：
   如果您的 OpenClaw 版本支持显式的 Slash Commands（斜杠命令），它通常会自动将 `workflows` 目录下的 Markdown 文件名读取为快捷命令。该脚本对应的是 `/auto-inquiry`。

## 3. 如何使用该技能
以后无论客户是在微信里发了一长串文字，还是发了一封格式混乱的邮件，您只需要复制那段话，并直接向您的 OpenClaw 发送以下指令即可：

> **您对 OpenClaw 发送的话术模板：**
> “请使用 `/auto-inquiry` 技能，帮我处理下面这个客户询价：
> [这里粘贴客户的长篇大论，例如：老板好，麻烦帮我报个价，需要 FL-08 那个型号50个，还有那个特殊接头（你懂的）要100个...]”

## 4. OpenClaw 的自动执行流程展示
收到上述指令后，您的 OpenClaw 会“自动触发”以下连招（您可以在屏幕上看它表演）：
1. 💡 **智能提取**：OpenClaw 会明白：“哦！我要提取 FL-08（50个）和 特殊接头（100个）。”
2. 💾 **生成临时文件**：OpenClaw 自动将提取到的数据写入系统隐藏的临时 JSON 列表中。
3. ⚙️ **执行引擎命令**：OpenClaw 自动运行终端命令 `node server/scripts/ai_auto_quote.js /tmp/xxx.json`，让本地数据库去运算。
4. 📊 **分析结果并向您汇报**：最后，OpenClaw 会根据脚本返回的结果直接回复您：
   - “✅ 已成功在系统中为您生成草稿报价单 **Q20260322-AIQ6AH**，总金额 **¥XXXX**。”
   - “⚠️ 注意：客户说的**‘特殊接头（你懂的）’**在系统别名库中没有找到对应映射！这是一个未匹配的异常项，请您前往**‘产品别名管理’**板块手动将它绑定到某个标准产品上，或补充进报价单！”

## 5. 日常维护提醒
这个技能非常依赖于系统右上角的**“产品别名管理”**板块。
当 AI 告诉您有“未匹配/找不到”的异常产品时，只要您去后台将那个奇奇怪怪的叫法（别名）**手动增加映射**到正确的标准库产品上，下次客户再说同样的话，AI 就能 100% 自动给它算进报价单里了！系统将越用越聪明！
