# 飞书报价机器人使用指南

## 工作流概览

```
飞书发送报价Excel
        ↓
机器人自动报价（5条匹配规则）
        ↓
返回带价格的Excel文件
        ↓
人工填写未匹配产品的价格
        ↓
发回反馈文件给机器人
        ↓
自动更新标准库 + 重复编码检查
```

---

## 使用方法

### 1. 报价流程

**发送消息格式：**
```
报价
产品列: 1
数量列: 3
```
**并附上 Excel 报价单附件**

列号说明：A=0, B=1, C=2, D=3, E=4, F=5...

### 2. 更新标准库流程

**发送消息格式：**
```
更新标准库
产品列: 0
价格列: 5
标注列: 10
```
**并附上填好价格的反馈 Excel 文件**

---

## 5条匹配规则（按优先级）

1. **规则一**：第五位 1↔2 双向互换
2. **规则五**：前五位编码映射表（如 30411↔20411）
3. **规则二**：移除末尾 T 后匹配
4. **规则三**：去掉 `*` 及后续内容直到下一个 `-`
5. **规则四**：O/o → 0 字符容错替换

## 常见问题

**Q: 匹配率低怎么办？**
A: 查看日志文件，检查未匹配的编码，考虑：
1. 新增规则五的前五位映射
2. 人工填写价格后发回更新标准库

**Q: Excel文件读不出来？**
A: 确保文件未被其他程序打开，且是标准 .xlsx 格式

**Q: 怎么扩展规则？**
A: 编辑 `matcher.py` 中的 `CodeMatcher` 类，添加新的匹配方法
